    Range1dArray Class

    A Range1d array is a set of intervals, such as occur when a line is clipped to a (nonconvex) polygon

    Methods

    Name Description
    differenceSorted(dataA: Range1d[], dataB: Range1d[]): Range1d[] Static Intersect intervals in two pre-sorted sets.
    getBreaks(data: Range1d[], result?: GrowableFloat64Array): GrowableFloat64Array Static return an array with all the low and high values of all the ranges.
    intersectSorted(dataA: Range1d[], dataB: Range1d[]): Range1d[] Static
    isSorted(data: Range1d[], strict: boolean = true): boolean Static Test if the low,high values are sorted with no overlap.
    paritySorted(dataA: Range1d[], dataB: Range1d[]): Range1d[] Static
    simplifySortParity(data: Range1d[], removeZeroLengthRanges: boolean = false): void Static
    simplifySortUnion(data: Range1d[], removeZeroLengthRanges: boolean = false): void Static Cleans up the array, compressing any overlapping ranges.
    sort(data: Range1d[]): void Static Uses the Range1d specific compare function for sorting the array of ranges
    sumLengths(data: Range1d[]): number Static sum the lengths of all ranges
    testParity(data: Range1d[], value: number): boolean Static test if value is "in" by parity rules.
    testUnion(data: Range1d[], value: number): boolean Static test if value is "in" by union rules.
    unionSorted(dataA: Range1d[], dataB: Range1d[]): Range1d[] Static
